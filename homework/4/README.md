## 3.13 确定四种化合物在混合物中所占质量百分比

+ 解题思路:

1. 先列出方程组 将元素质量带入每个化合物得到总共四个方程 将方程组归为矩阵 元素依次为碳氢氮氧
2. 由于numpy计算矩阵线性方程组时只能满秩行列式 因此每次将其中一种元素替换为[1, 1, 1, 1]的归一方程 
3. 重复四次后得到各化合物的物质的量比例 乘上相对分子质量取平均即可

+ 计算结果:

硝基苯: 苯胺: 氨基丙酮: 乙醇 = 39.5%: 14.9%: 23.3% : 22.3%
